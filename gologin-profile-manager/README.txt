# GoLogin Profile Manager

## Yêu cầu
- Windows 10/11 (64-bit)
- Node.js v18+ (tải từ https://nodejs.org)

## Cách chạy
1. Cài đặt Node.js nếu chưa có
2. Double-click file "Start.bat"

## Cấu trúc thư mục
- data/ - Dữ liệu người dùng (profiles, settings, metadata)
- data/profiles/ - Browser profiles (tạo tự động)
- orbita-browser-*/ - Chromium browser

## Cập nhật phiên bản mới
- Copy đè tất cả NGOẠI TRỪ thư mục data/
- Thư mục data/ chứa dữ liệu người dùng, KHÔNG được xóa

## Lưu ý
- Ứng dụng sử dụng cổng 3000
- Không di chuyển các file ra khỏi thư mục này
